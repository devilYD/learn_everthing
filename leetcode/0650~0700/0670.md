# 最大交换
## 题目
给定一个非负整数，你至多可以交换一次数字中的任意两位。返回你能得到的最大值。
```
示例:
输入: 2736
输出: 7236
解释: 交换数字2和数字7。
```
## 代码
```cpp
class Solution {
public:
    int maximumSwap(int num) {
        string s = to_string(num);
        int n = s.size();
        int beg=0;
        int max;
        char c;
        while(s[beg]>=s[beg+1]&&beg<n) beg++;
        if(beg == n) return num;
        max=beg;
        for(int i=beg;i<n;i++)
        {
            if(s[i]>=s[max]) max=i;
        }
        for(int i=0;i<max;i++)
        {
            if(s[i]<s[max])
            {
                n=i;
                break;
            }
        }
            c=s[max];
            s[max]=s[n];
            s[n]=c;
        
        return stoi(s);
    }
};
```
